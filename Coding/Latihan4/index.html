Laporan Praktikum: Dasar Pemrograman Web

Pada pertemuan kali ini, saya telah mempelajari bagaimana membuat struktur dokumen web yang benar menggunakan HTML. HTML atau HyperText Markup Language berfungsi untuk menyusun kerangka sebuah halaman web agar informasi dapat tersampaikan dengan jelas kepada pengguna.

Selain HTML, saya juga mulai mengenal CSS (Cascading Style Sheets). Dengan CSS, saya dapat mengatur tampilan visual seperti warna, dan font. Hal ini sangat membantu dalam membuat halaman web menjadi lebih menarik dan profesional.

Demikzian laporan singkat ini saya buat sebagai bagian dari tugas praktikum mata kuliah Pengenalan Pemrograman. Terima kasih atas bimbingan yang telah diberikan.

Mengatur Padding

Padding adalah ruang kosong antara konten dan batas (border) elemen, atau bantalan material yang bertujuan mengatur tata letak, estetika, kenyamanan, dan keamanan. Dalam desain web, ia membuat teks/gambar tidak mepet ke tepi, sementara pada pakaian atau barang, ia memberikan bantalan pengaman.
